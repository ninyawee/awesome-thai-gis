# Awesome Thai GIS

Free downloadable resources for doing GIS in Thailand

รวมเว็บไซต์ดาวน์โหลดไฟล์ GIS ของประเทศไทย (ฟรี)

## Static Map Data

### [Thai ECT Election Map 2023](https://github.com/KittapatR/Thai-ECT-election-map-66) - Electoral district boundaries
- **Organization**: KittapatR (Community)
- **Data types**: ESRI ShapeFile, SVG, GeoJSON
- **Coverage**: Thailand electoral districts, municipalities, subdistricts
- **Update frequency**: Electoral cycle

### [Thai Province Data](https://github.com/kongvut/thai-province-data) - Administrative divisions metadata
- **Organization**: kongvut (Ratchaphruek Hospital Public Company Limited, Khon Kaen)
- **Data types**: CSV, JSON, SQL, XLSX, XML with JSON APIs
- **Coverage**: Thailand provinces (จังหวัด), districts (อำเภอ), sub-districts (ตำบล) with coordinates and zip codes
- **Update frequency**: Maintained with changelog, last major update v2
- **Notable features**: Ready-to-use JSON APIs, JSON Schema validation, Docker support, React demo

### [NETH's E-Portfolio](https://csuwan.weebly.com/360436343623360936603650362736213604--download.html) - Administrative and infrastructure data
- **Organization**: NETH
- **Data types**: Province, Amphoe, Tambon, Village boundaries; Schools; Roads; Land lots; Satellite imagery
- **Coverage**: Thailand
- **Update frequency**: Unknown

### [HDX Thailand Administrative Boundaries](https://data.humdata.org/dataset/thailand-administrative-boundaries) - Administrative boundaries
- **Organization**: Humanitarian Data Exchange
- **Data types**: Province (Polygon), Amphoe (Polygon), Tambon (Line)
- **Coverage**: Thailand
- **Update frequency**: Last update 2017

### [Thailand Subnational Administrative Boundaries (COD-AB)](https://data.humdata.org/dataset/cod-ab-tha) - Administrative boundaries (ประเทศ, จังหวัด, อำเภอ, ตำบล)
- **Organization**: กรมแผนที่ทหาร (RTSD), OCHA ROAP, ITOS
- **Data types**: Administrative boundary polygons at national, province, district, and sub-district levels
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [ข้อมูลที่ตั้งและสภาพทั่วไปของหมู่บ้านใน 76 จังหวัด](https://gdcatalog.go.th/th/dataset/gdpublish-gis-01) - Village location and general information
- **Organization**: Government Data Catalog (GD Catalog)
- **Data types**: Village location data for 76 provinces
- **Coverage**: Thailand (76 provinces)
- **Update frequency**: Periodic

### [ระบบฐานข้อมูลภูมิสารสนเทศทรัพยากรธรณี](https://gis.dmr.go.th/DMR-GIS/) - Geological resource data (ธรณีวิทยา)
- **Organization**: กรมทรัพยากรธรณี (DMR)
- **Data types**: Rock units, minerals, faults, coastal data, and more (โปรดดู Metadata และ Data dictionary ของแต่ละข้อมูลประกอบการใช้งาน)
- **Coverage**: Thailand
- **Update frequency**: Varies by dataset

### [ดินออนไลน์ (Din Online)](https://dinonline.ldd.go.th/) - Soil and land use data service
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Soil data, land use data (ตั้งแต่ พ.ศ. 2543 - ปัจจุบัน)
- **Coverage**: Thailand
- **Update frequency**: Regularly updated
- **Notable features**: Registration required; data available on request

### [กลุ่มชุดดิน (Soil Group)](https://gdcatalog.go.th/dataset/gdpublish-oss01) - Soil group classification
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Soil groups classified by similar soil properties
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [ชุดดิน (Soil Series)](https://lddcatalog.ldd.go.th/dataset/ldd_11_01) - Soil series data (Soil Taxonomy)
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Soil series data classified by Soil Taxonomy
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [การชะล้างพังทลายของดิน (Soil Erosion)](https://gdcatalog.go.th/dataset/gdpublish-lpd06) - Soil erosion data
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Soil erosion assessment data
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [EarthEnv-DEM90](https://www.earthenv.org/DEM) - Digital Elevation Model (~90m resolution)
- **Organization**: Robinson et al. (2014)
- **Data types**: DEM raster (~90m spatial resolution)
- **Coverage**: Global (including Thailand)
- **Update frequency**: Static dataset

### [Viewfinder Panorama](http://viewfinderpanoramas.org/dem3.html) - Digital elevation models
- **Organization**: Viewfinder Panoramas
- **Data types**: DEM 1″, 3″, 15″ (raster)
- **Coverage**: Thailand, global
- **Update frequency**: Last update 2014

### [OpenTopography](https://opentopography.org/) - Multiple DEM datasets
- **Organization**: OpenTopography
- **Data types**: Multiple DEM sources (SRTM, ALOS, Copernicus, etc.) with DOI citations
- **Coverage**: Global (including Thailand)
- **Update frequency**: Varies by dataset
- **Notable features**: Multiple DEM sources available; cite using DOI provided on the website

### [เส้นชั้นความลึกพื้นท้องทะเล (Bathymetry)](https://gdcatalog.go.th/dataset/gdpublish-bathymetry) - Seafloor depth contours
- **Organization**: กรมทรัพยากรธรณี (DMR)
- **Data types**: Bathymetric contour data
- **Coverage**: Thailand coastal and marine areas
- **Update frequency**: Periodic

### [ป่าไม้ถาวร (Permanent Forest)](https://data.thailand.opendevelopmentmekong.net/th/dataset/thailand-permanent-forest) - Permanent forest boundaries
- **Organization**: กรมป่าไม้ (RFD)
- **Data types**: Permanent forest area boundaries
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [ป่าสงวนแห่งชาติ (National Reserved Forest)](https://data.forest.go.th/dataset/reserve_forest) - National reserved forest data
- **Organization**: กรมป่าไม้ (RFD)
- **Data types**: National reserved forest boundaries
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [Protected Planet - Thailand](https://www.protectedplanet.net/country/THA) - World Database on Protected Areas (WDPA)
- **Organization**: UNEP-WCMC and IUCN
- **Data types**: Protected areas and Other Effective Area-based Conservation Measures (OECM)
- **Coverage**: Thailand
- **Update frequency**: Regular updates
- **Citation**: UNEP-WCMC and IUCN (2025), Protected Planet: WDPA and WD-OECM

### [การใช้ที่ดินปัจจุบัน (Current Land Use)](https://gdcatalog.go.th/dataset/gdpublish-lpd01) - Current land use data
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Current land use classification
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [ความเหมาะสมของที่ดินสำหรับการปลูกพืชเศรษฐกิจ](https://gdcatalog.go.th/dataset/gdpublish-lpd02) - Land suitability for economic crops
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Land suitability assessment for economic crops
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [สำมะโนที่ดิน (Land Census)](https://lddcatalog.ldd.go.th/dataset/osm02) - Land census data
- **Organization**: กรมพัฒนาที่ดิน (LDD)
- **Data types**: Land census data (available for some sub-districts and provinces)
- **Coverage**: Partial (บางตำบล บางจังหวัด)
- **Update frequency**: Periodic

### [ชั้นคุณภาพลุ่มน้ำ (Watershed Classification)](https://gdcatalog.go.th/dataset/gdpublish-wcms2) - Watershed quality classification
- **Organization**: สำนักงานนโยบายและแผนทรัพยากรธรรมชาติและสิ่งแวดล้อม (ONEP)
- **Data types**: Watershed quality classification data
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [ขอบเขตลุ่มน้ำหลัก 22 ลุ่ม และลุ่มน้ำสาขา](http://www.onwr.go.th/?page_id=9893) - Main and sub-basin boundaries
- **Organization**: สำนักงานทรัพยากรน้ำแห่งชาติ (ONWR)
- **Data types**: 22 main river basin boundaries and sub-basin boundaries
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [ชั้นข้อมูลแหล่งน้ำ (FGDS)](https://www.data.go.th/th/dataset/item_60fac91a-01c5-43da-b429-19e145a1ce7a) - Water source data
- **Organization**: กรมทรัพยากรน้ำ
- **Data types**: Water source fundamental geographic data
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [บ่อน้ำบาดาล (Groundwater Wells)](https://dgr.gdcatalog.go.th/dataset/dgr) - Government groundwater well data
- **Organization**: กรมทรัพยากรน้ำบาดาล (DGR)
- **Data types**: Government groundwater well locations and data
- **Coverage**: Thailand
- **Update frequency**: Periodic

### [แผนที่น้ำบาดาลรายจังหวัด](https://datum.dgr.go.th/mapgroundwater/) - Provincial groundwater maps
- **Organization**: กรมทรัพยากรน้ำบาดาล (DGR)
- **Data types**: Groundwater maps by province
- **Coverage**: Thailand (by province)
- **Update frequency**: Periodic

### [Humanitarian OpenStreetMap Team (HOT) - Thailand](https://data.humdata.org/organization/hot?q=thailand&sort=last_modified%20desc&ext_page_size=25) - OpenStreetMap extracts for Thailand
- **Organization**: Humanitarian OpenStreetMap Team (HOT)
- **Data types**: Roads, buildings, waterways, railways, points of interest, financial services, and more
- **Coverage**: Thailand
- **Update frequency**: Regular updates

### [Open Government Data of Thailand](https://data.go.th/) - Thailand open government data portal
- **Organization**: Government of Thailand
- **Data types**: Various open datasets from Thai government agencies
- **Coverage**: Thailand
- **Update frequency**: Varies by dataset

### [Natural Earth Data](http://www.naturalearthdata.com/downloads/) - Natural and cultural features
- **Organization**: Natural Earth
- **Data types**: Administrative boundaries, coastlines, roads, railroads, airports, ports, islands, elevation
- **Coverage**: Thailand, global
- **Update frequency**: Last update 2017

### [U Penn's International GIS data](https://guides.library.upenn.edu/c.php?g=475518&p=3254775) - Academic GIS resources
- **Organization**: University of Pennsylvania
- **Data types**: Various
- **Coverage**: Thailand, Asian countries
- **Update frequency**: Last update 2017

## Dynamic Map Data

### [WorldClim](https://worldclim.org/data/index.html) - Global climate and weather data
- **Organization**: WorldClim
- **Data types**: Historical and future climate scenarios (temperature, precipitation, bioclimatic variables)
- **Coverage**: Global (including Thailand)
- **Update frequency**: Periodic releases

### [CHIRPS Rainfall Data](https://www.chc.ucsb.edu/data/chirps3) - Satellite rainfall estimates
- **Organization**: Climate Hazards Center, UC Santa Barbara
- **Data types**: Rainfall estimates from rain gauge and satellite observations (daily, monthly, yearly)
- **Coverage**: Global (including Thailand)
- **Update frequency**: Regular updates

### [CHRS Data Portal](https://chrsdata.eng.uci.edu/) - Satellite precipitation data
- **Organization**: Center for Hydrometeorology and Remote Sensing, UC Irvine
- **Data types**: Satellite-based precipitation data
- **Coverage**: Global (including Thailand)
- **Update frequency**: Regular updates

### [ปริมาณน้ำฝน/น้ำท่า กรมชลประทาน](https://water.rid.go.th/hydrology/) - Rainfall and streamflow data
- **Organization**: ส่วนอุทกวิทยา สำนักบริหารจัดการน้ำและอุทกวิทยา กรมชลประทาน (RID)
- **Data types**: Rainfall, streamflow, and sediment data from regional centers (1-8)
- **Coverage**: Thailand
- **Update frequency**: Regular updates
- **Notable features**: Data must be downloaded from each regional center individually

## Applications

### [Thailand Flood Realtime Dashboard](https://disaster.gistda.or.th/dashboard) - Flood monitoring application
- **Organization**: GISTDA using [Vallaris Maps](#vallaris-maps---cloud-gis-platform)
- **Purpose**: Real-time flood monitoring and water level stations
- **Coverage**: Thailand
- **Update frequency**: Real-time

### [National Thai Water Dam Information](https://nationalthaiwater.onwr.go.th/dam) - Dam and water resource monitoring (สถานการณ์น้ำประเทศไทย)
- **Organization**: Office of National Water Resources (ONWR)
- **Purpose**: Water situation monitoring, dam levels, and water resource data visualization
- **Coverage**: Thailand
- **Update frequency**: Real-time

### Land Parcel Mapping - Government land data visualization
- **Organization**: Various government agencies
- **Purpose**: Land plot visualization for provinces near Bangkok
- **Coverage**: Bangkok metropolitan area
- **Update frequency**: Varies

### Active Fire Monitoring - Satellite fire detection
- **Organization**: Various
- **Purpose**: Heat point tracking from satellite observations
- **Coverage**: Thailand
- **Update frequency**: Daily

### Population Visualization Map (2020) - Population density mapping
- **Organization**: Various
- **Purpose**: Machine learning analysis of satellite imagery for population density
- **Coverage**: Thailand
- **Update frequency**: Census cycle

### COVID-19 Mapping Applications - Pandemic tracking
- **Organization**: Various government agencies
- **Purpose**: Safe transit points and case visualization
- **Coverage**: Thailand
- **Update frequency**: During pandemic period

## Tools

### [Overpass Turbo](https://overpass-turbo.eu/) - OpenStreetMap query tool
- **Purpose**: Extract GeoJSON data from OpenStreetMap
- **Type**: Web-based query interface

### [mapshaper](http://mapshaper.org/) - GIS format converter
- **Purpose**: Convert between GeoJSON, KML, CSV, DBF, and Shapefile
- **Type**: Web-based tool

### [MyGeodata Cloud](https://mygeodata.cloud/) - GIS data converter
- **Purpose**: Convert between various GIS formats
- **Type**: Web-based service

### [kepler.gl](https://kepler.gl/) - Geospatial visualization
- **Purpose**: Interactive geospatial data visualization
- **Type**: Uber's open source web tool

### [GeoJSONLint](http://geojsonlint.com/) - GeoJSON validator
- **Purpose**: Validate and visualize GeoJSON files
- **Type**: Web-based validator

### [Project Linework](http://www.projectlinework.org/) - Cartographic resources
- **Purpose**: Handcrafted lineworks for cartography
- **Type**: Design resource
